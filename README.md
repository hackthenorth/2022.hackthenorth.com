### Running locally

First, clone the repo

```
$ git clone https://github.com/hackathon/hackthenorth.com.git
$ cd hackthenorth.com
```

Then install dependencies and get it running

```
$ npm install
$ npm start
```

Navigate to `localhost:3000` in your browser and give it a look 👀
